Emotion Based Music Recommendation System

Face-Tune

This project aims to recognize human emotions using facial expression analysis and provide real-time music recommendations based on the detected emotion. It utilizes a convolutional neural network (CNN) model trained to classify emotions such as happiness, sadness, surprise, disgust, anger, and neutrality. The facial expression recognition is performed using OpenCV and Haar cascades for face detection. Have a look at the model: Face Tune

Attachment Chatting Bot

Attachment Bot

Developed a system to upload, extract, and process text from PDF, Word, and Excel documents using libraries like PyPDF2, docx, and openpyxl. Implemented text chunking and generated embeddings using RecursiveCharacterTextSplitter and Google Generative AI models, stored and searched efficiently with FAISS. Built an interactive GUI with Tkinter that allows users to query document contents, providing detailed answers through a conversational AI model leveraging Google Generative AI. Have a look at the model: Attacment bot
